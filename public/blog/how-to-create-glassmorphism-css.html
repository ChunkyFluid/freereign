What is glassmorphism?

Glassmorphism (popularised by macOS Big Sur and Windows 11's "Acrylic" material) layers a semi-transparent surface over busy content and blurs whatever sits behind it. The result reads as a sheet of frosted glass. Three ingredients make it work: translucency (a background with alpha), a backdrop blur, and a subtle light border that catches the "edge" of the glass.

The core CSS

Here's the minimal, production-ready recipe. Every glassmorphism effect is a variation on these five lines:

.glass {
  /* 1. translucent fill — keep alpha low (0.08–0.2) */
  background: rgba(255, 255, 255, 0.12);

  /* 2. blur what's behind it (Safari needs the -webkit- prefix) */
  -webkit-backdrop-filter: blur(12px) saturate(160%);
  backdrop-filter: blur(12px) saturate(160%);

  /* 3. a faint light border for the glass edge */
  border: 1px solid rgba(255, 255, 255, 0.25);

  /* 4. soft depth */
  border-radius: 16px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.25);
}

Why each line matters:

  • background: rgba(...) — the glass tint. Low alpha keeps it readable without hiding the colour behind it. Use black tints on light pages, white tints on dark/colourful ones.
  • backdrop-filter: blur() — the magic. saturate(160%) pumps the colour behind the glass so it doesn't look washed out.
  • The border — a 1px semi-transparent white edge simulates the refractive rim of glass and stops the panel from blending into the background.
  • Shadow — lifts the panel off the page so it reads as a floating surface.

Browser support & the Safari prefix

backdrop-filter is supported in all modern browsers — Chrome, Edge, Firefox, and Safari — but Safari and iOS still require the -webkit-backdrop-filter prefix. Always declare both, prefix first. If you need a fallback for very old browsers, bump the background alpha higher so the panel is still legible without any blur:

@supports not (backdrop-filter: blur(1px)) {
  .glass { background: rgba(255, 255, 255, 0.6); }
}

The gotchas nobody warns you about

1. It needs a busy background

The single most common mistake: placing a glass card on a plain solid colour. With nothing interesting behind it, there's nothing to blur, and the effect collapses into a flat translucent box. Glass only looks like glass over an image, a gradient, or other colourful content.

2. Watch performance

backdrop-filter is GPU-accelerated but expensive — the browser re-samples and blurs everything behind the element. Keep the number of simultaneously visible glass surfaces small, never animate the blur radius, and test on a real low-end phone before shipping.

3. Mind the contrast (accessibility)

Translucent panels can wreck text contrast depending on what scrolls behind them. Verify text meets WCAG AA (4.5:1) against the worst-case background, not just the screenshot you designed against. A slightly higher background alpha or a text shadow is often the fix.

Copy-paste recipes

Glass card

.glass-card {
  background: rgba(255, 255, 255, 0.12);
  -webkit-backdrop-filter: blur(14px) saturate(160%);
  backdrop-filter: blur(14px) saturate(160%);
  border: 1px solid rgba(255, 255, 255, 0.22);
  border-radius: 18px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.25);
  padding: 28px;
}

Sticky glass navbar

.glass-nav {
  position: sticky; top: 0;
  background: rgba(15, 15, 20, 0.55);
  -webkit-backdrop-filter: blur(16px);
  backdrop-filter: blur(16px);
  border-bottom: 1px solid rgba(255, 255, 255, 0.08);
}

Glass modal overlay

.glass-modal {
  background: rgba(30, 30, 40, 0.45);
  -webkit-backdrop-filter: blur(20px) saturate(180%);
  backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.18);
  border-radius: 20px;
}

Glassmorphism in Tailwind CSS

Tailwind ships the backdrop utilities you need out of the box:

<div class="bg-white/10 backdrop-blur-md backdrop-saturate-150
            border border-white/20 rounded-2xl shadow-lg p-7">
  Frosted glass
</div>

For an exact blur or saturation value, use arbitrary values: backdrop-blur-[14px] and backdrop-saturate-[160%].

FAQ

Does glassmorphism hurt performance?

It can. backdrop-filter is one of the more expensive effects because the browser blurs everything behind the element on every frame. Limit how many glass surfaces are visible at once, don't animate the blur, and test on low-end mobile.

Does Safari support backdrop-filter?

Yes, with the -webkit-backdrop-filter prefix. Declare both the prefixed and unprefixed properties and it works across Safari, iOS, Chrome, Edge, and Firefox.

How do I do glassmorphism in Tailwind?

Combine backdrop-blur-md, backdrop-saturate-150, a translucent bg-white/10, a faint border border-white/20, and rounded-2xl shadow-lg.

Why does mine look like a flat box?

There's nothing colourful behind it to blur. Place the glass over an image or gradient — on a solid background the effect disappears.
